# Project Overview

This repository contains an **Interactive Excel Sales Dashboard** created using the Super Store dataset.  
The dashboard is designed to provide a complete analytical view of sales performance, customer segments, regional insights, and category-wise patterns.

It uses dynamic charts, slicers, and clean visualization to help users analyze trends and make data-driven decisions.



# Tools & Techniques Used

- **Microsoft Excel**
- **Power Query** (Data Cleaning & Calendar Table)
- **Pivot Tables / Pivot Charts**
- **Slicers & Timelines**
- **Conditional Formatting**
- **Data Modeling**
- **Advanced Chart Formatting**
- **KPI Cards & Icons**
